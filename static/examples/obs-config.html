üìã Setup Instructions

  1. Configure your overlay using the checkboxes above
  2. Copy the generated URL by clicking the "Copy URL" button
  3. Open OBS Studio and add a new Browser Source
  4. Paste the URL into the Browser Source settings
  5. Set dimensions: Width: 1920, Height: 1080 (or your stream resolution)
  6. Make sure your Nuzlocke tracker is running on http://localhost:5173
  7. Start the standalone server on http://localhost:5174
  8. Your overlay will automatically sync with your tracker data!
